# TZTV-iOS
个人iOS项目

自从做了Java就没写过iOS了，趁最近有空把以前做的项目再拿出来重构一下。


## 重构计划

- 用YJRouter来统一管理页面的跳转。
- 重新封装网络请求。
- 用pod加载本地私有库，抽取公共基础代码。
- 采用Java编写后台服务。
